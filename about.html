  • About Child-View Project

An Egocentric Visual Datasets generated in a Virtual Environment.

Child-View project is a project of egocentric visual datasets that are generated in a virtual environment named 'TDW', which is generated by Massachussets Institution of Technology (MIT).
Our goal is to share our datasets and give introductions of how to build new datasets to help researchers to start deep learning.

Cognitive Psychology behind Child-View

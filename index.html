    An Egocentric Visual Deep Learning Database Built from Navigating Virtual Environments

A visual deep-learning datasets inspired by cognitive development

Many visual learning databases rely on millions of independent images to train their algorithms and models. But these databases far exceed the standard visual exposure a typical human toddler makes, and provide minimal improvements in classification performance. Because children learn to perceive within constrained visual environments, Child-view mimics this visual learning environment with a spatiotemporally correlated array of images from the same virtual environment, enabling efficient learning even with smaller datasets.

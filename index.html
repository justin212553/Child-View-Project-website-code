    Egocentric Visual Deep Learning Database Built in Virtual Environments

Visual Deep Learning Datasets

Egocentric visual datasets built in virtual environments, with spatial and time information

Child-View project is a project of egocentric visual datasets that are generated in a virtual environment named 'TDW', which is generated by Massachussets Institution of Technology (MIT). Our goal is to share our datasets and give introductions of how to build new datasets to help researchers to start deep learning.

How to build your own datasets

Make your own dataset with TDW virtual environments with our introductions.

We provide a step-by-step documentataion of TDW to build your own virtual environment egocentric datasets for your visual deep learning project.
